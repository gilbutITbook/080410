# 노드 버전 변경

모든 예제는 Node.js 16.20.0 버전을 기준으로 작성되었습니다.

`nvm`을 사용해 프로젝트에서 사용하는 16.20.0 버전을 설치합니다.

```bash
nvm install 16.20.0
```

nvm을 사용해 다음의 명령어를 실행하시면 16.20.0 버전을 사용하실 수 있습니다.

```bash
nvm use
```

만약, 위의 명령어가 실행되지 않는다면, 아래의 명령어를 실행해 보세요.

```bash
nvm use 16.20.0
```
